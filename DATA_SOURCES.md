# Data Sources

## Sample Documents

10 synthetic financial documents generated by `create_sample_data.py`:
- Annual reports, financial statements, risk assessments
- Investment strategies, regulatory compliance, market analysis
- Revenue/cost analysis, balance sheets, cash flow statements

**Rationale**: Provides comprehensive financial document types for testing without copyright issues.

## Production Data Sources

For production, use real financial documents from:

- **SEC EDGAR**: https://www.sec.gov/edgar/ (10-K, 10-Q filings)
- **Investor Relations Pages**: Annual reports, earnings releases
- **FRED**: https://fred.stlouisfed.org/ (Economic indicators)

## Data Processing

- **Format**: PDF and text files
- **Chunking**: 1000 chars with 200 char overlap
- **Total**: 10 documents, ~50-100 chunks
- **Storage**: ChromaDB vector database

## Adding Documents

Place documents in `data/` directory and run:
```bash
python main.py --setup
```
